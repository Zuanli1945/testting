## Deskripsi Proyek
Website interaktif untuk **membuat dan mengerjakan quiz matematika dasar**.  
Tujuannya membantu siswa meningkatkan kemampuan matematika dasar melalui latihan soal yang menyenangkan dan mudah diakses.

## Fitur
- Membuat quiz dengan operasi dasar: penjumlahan, pengurangan, perkalian, pembagian, dan akar
- Menampilkan skor dan jawaban yang benar setelah selesai  
- Antarmuka user-friendly dan responsif di berbagai perangkat  

## Teknologi
- HTML  
- CSS  
- JavaScript
